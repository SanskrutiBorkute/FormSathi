1. Project Overview
=========================================================

FormSathi is an AI-powered web application that helps users understand, validate, and analyze various forms and documents. The system extracts text using OCR technology, identifies important fields, detects missing information, validates user inputs, and provides AI-generated guidance to simplify the form-filling process.

The project consists of two major parts:

• Frontend (React + Vite)
• Backend (Flask + Python)

The frontend provides a modern and interactive user interface, while the backend performs OCR, AI processing, validation, and report generation.

4. Clone the Project
=========================================================

Clone the repository from GitHub.

git clone <repository-url>

cd FormSathi

=========================================================
5. Backend Setup
=========================================================

Step 1

Open the backend folder.

cd backend

Step 2

Create a virtual environment.

python -m venv .venv

Step 3

Activate the environment.

Windows

.venv\Scripts\activate

Step 4

Install dependencies.

pip install -r requirements.txt

Step 5

Run the Flask server.

python app.py

Backend will start on

http://localhost:8080

=========================================================
6. Frontend Setup
=========================================================

Open another terminal.

Go inside frontend.

cd frontend

Install packages.

npm install

Run React.

npm run dev

The frontend will open at

http://localhost:5173

=========================================================
7. Running the Complete Project
=========================================================

Start Backend
