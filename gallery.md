# 📸 FormSathi Project Gallery

This gallery showcases the major interfaces of **FormSathi – AI Powered Multilingual Smart Form Assistant**.

---

## 1. Login Page

Users can securely log in to FormSathi using their registered email and password. The interface also supports multilingual language selection.

![Login Page](screenshots/Login%20page.png)

---

## 2. Registration Page

New users can create an account by entering their basic details before accessing the AI-powered platform.

![Register Page](screenshots/Register%20Page.png)

---

## 3. Home Dashboard

The home dashboard allows users to upload forms, scan documents using the camera, and start AI-powered form processing.

![Home Page](screenshots/Home%20page.png)

---

## 4. AI Form Analysis Workspace

The uploaded form is processed using OCR and AI. Users can edit extracted fields, receive intelligent guidance, validate information, and interact with the Smart Form Assistant.

![Form Filling](screenshots/Form%20Filling.png)

---

## 5. Analytics Dashboard

The analytics dashboard provides insights into OCR accuracy, validation success, completed fields, pending errors, multilingual coverage, and overall system performance.

![Analytics Page](screenshots/Analytics%20Page.png)

---

## 6. History & Analysis

Users can review previous uploads, OCR accuracy trends, category distribution, activity logs, and document history for better tracking and monitoring.

![History and Analysis](screenshots/History%20and%20Analysis%20.png)
